功能特性

你需要的，都在这里

从查词到复习，Voca 覆盖单词学习的完整流程。

⚡
全局划词翻译
选中任意文字，双击 Ctrl+C 即可触发翻译悬浮窗，无需切换应用，不打断当前工作流。
📷
OCR 截图翻译
按下 Ctrl+Shift+O，框选屏幕任意区域，自动识别文字并翻译。视频字幕、截图文字全都能查。
⭐
一键收藏生词
在悬浮翻译窗口点击收藏，单词自动存入生词本，支持短语和长句，不限字数。
🗂
间隔重复闪卡
基于 SM-2 算法，根据你的记忆情况智能安排复习间隔，有到期提醒，学了不白学。
📚
多生词本管理
按主题或来源创建多个生词本，内置 IELTS / TOEFL / GRE 预设词表，一键导入并自动翻译。
📊
学习统计
打卡连续天数、今日复习进度、掌握程度分布一目了然，支持设置每日目标。
🔍
全局搜索
跨所有生词本搜索单词或翻译，即使忘了存在哪本，也能快速找到。
↗
Anki 导出
支持将生词本导出为 Anki 兼容的 TSV 格式，已有 Anki 使用习惯的用户可无缝衔接。
🌐
多翻译引擎
支持 Google 翻译、DeepL 和 MyMemory，按需切换，支持多语言互译。
使用方式

查到就收，
随时复习

1
选中文字，双击 Ctrl+C
在任何应用里选中单词或句子，悬浮翻译窗口自动弹出。
2
点击 ⭐ 收藏到生词本
单词连同翻译一起保存，支持存入指定生词本。
3
用闪卡复习，巩固记忆
系统根据 SM-2 算法自动安排复习时间，越来越熟的词间隔越来越长。
